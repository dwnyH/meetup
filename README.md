#Meet-up
Meet-up API와 다음 지도를 이용한, 지역기반 모임 웹 어플리케이션입니다.

## Setup

Install dependencies

```sh
$ yarn install (or npm install)
```

## Development

```sh
$ yarn dev (or npm run dev)
# visit http://localhost:8080
```

## Features

- 지도에서 원하는 지역을 선택하거나, 검색어를 입력해 지역 기반 모임을 검색 제공.
- 모임 즐겨찾기하는 기능 제공
